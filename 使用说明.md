# X (Twitter) 账户监控器 - 使用说明

## 功能特性

- 🔄 **多账户监控**：同时监控多个 X (Twitter) 账户
- ⏱️ **预设刷新频率**：1小时、5小时、30分钟、10分钟、1分钟，或自定义
- 📁 **按人分类文件夹**：每个账户独立文件夹，内含 images、videos、texts 子文件夹
- 🔄 **自动重命名**：文件重名时自动添加序号，不会覆盖
- 🖼️ **保持原始格式**：图片为 jpg/png，视频为 mp4
- 🖥️ **图形界面**：无需命令行，双击运行

## 快速开始（Windows）

### 方法一：直接运行 Python（推荐开发/测试）

1. 安装 Python 3.10+（勾选 "Add Python to PATH"）
2. 打开命令提示符，进入项目目录：
   ```
   cd x-twitter-monitor
   pip install -r requirements.txt
   python main.py
   ```

### 方法二：打包为 EXE（推荐日常使用）

1. 双击运行 `build_windows.bat`
2. 等待打包完成
3. 在 `dist/` 文件夹中找到 `X-Twitter-Monitor.exe`
4. 将 EXE 文件复制到任意位置，双击即可运行

## 使用步骤

### 1. 获取 X (Twitter) Cookie

1. 在浏览器中打开 [x.com](https://x.com) 并**登录**
2. 按 `F12` 打开开发者工具
3. 切换到 **Application（应用）** 标签
4. 左侧找到 **Cookies** → `https://x.com`
5. 复制以下值：
   - `auth_token`（必填）
   - `ct0`（必填）
   - `twid`（可选）

### 2. 连接认证

- 将复制的 Cookie 填入对应输入框
- 点击 **「连接认证」**
- 认证成功后，Cookie 会自动保存到 `cookies.json`，下次无需重新输入

### 3. 添加监控账户

- 输入要监控的 X 用户名（不含 @）
- 选择刷新频率（1小时、5小时等）
- 点击 **「添加监控」**
- 可重复添加多个账户

### 4. 设置下载目录

- 点击 **「浏览...」** 选择文件保存位置
- 默认保存在程序所在目录的 `downloads/` 文件夹

### 5. 开始监控

- 点击 **「全部启动」** 开始监控所有账户
- 点击 **「全部停止」** 停止所有监控

## 下载文件结构

```
下载目录/
├── 用户名A/
│   ├── images/          # 图片 (jpg/png)
│   │   ├── 123456789_photo_1.jpg
│   │   └── 123456789_photo_2.png
│   ├── videos/          # 视频 (mp4)
│   │   └── 123456789_video_1.mp4
│   └── texts/           # 推文文本 (json)
│       └── 123456789.json
└── 用户名B/
    ├── images/
    ├── videos/
    └── texts/
```

## 注意事项

- ⚠️ Cookie 有有效期，过期后需要重新获取
- ⚠️ 刷新频率不宜过短，建议至少 10 分钟，避免触发限制
- ⚠️ 请遵守 X 的使用条款，仅用于个人合法用途
- 💡 首次启动会记录最新推文作为基准，之后只下载新发布的推文
